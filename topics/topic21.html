Version Control & Git

Version control is a system that records changes to files over time so you can track history, revert to older versions, and collaborate with others. Git is the most popular version control tool used by developers worldwide.

Why Git?

- Tracks code history - Allows collaboration - Prevents file conflicts - Works offline - Supports branching and merging

Common Git Commands

git init – create repository git add . – stage changes git commit -m "message" – save changes git push – upload to server git pull – download updates git clone – copy a repository

GitHub

GitHub is a cloud platform where Git repositories are uploaded. It supports teams, project tracking, code review, and open-source collaboration.
