Networking Basics

Computer networking is the communication between multiple devices to share data and resources. Networks power the internet and modern communication.

Types of Networks

LAN: Local area (home, offices). WAN: Wide area (internet). MAN: Metropolitan area (city-wide).

Key Networking Concepts

IP Address: Unique identifier for devices. MAC Address: Hardware-level ID. Router: Directs network traffic. Switch: Connects devices within LAN. DNS: Converts domain names to IPs. Firewall: Blocks unauthorized access.

Protocols

HTTP / HTTPS – web communication FTP – file transfer SMTP – email TCP/UDP – transport layer

Why Networking Matters

Networking is the backbone of the internet, cloud systems, mobile networks, data centers, and cybersecurity.
