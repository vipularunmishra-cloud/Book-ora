Algorithms

1. What Are Algorithms?

An algorithm is a step-by-step method to solve a problem. Every computer program is built on algorithms, whether simple or complex.

2. Why Algorithms?

  • Improve speed of code
  • Reduce time and space complexity
  • Used in interviews
  • Foundation of machine learning, AI, and data science

3. Types of Algorithms

  • Sorting algorithms (Bubble, Merge, Quick)
  • Searching (Binary Search)
  • Greedy Algorithms
  • Dynamic Programming
  • Graph Algorithms (Dijkstra, BFS, DFS)

4. Example â€” Binary Search

int binarySearch(int arr[], int n, int x){
    int low = 0, high = n-1;
    while(low <= high){
        int mid = (low + high) / 2;
        if(arr[mid] == x) return mid;
        else if(arr[mid] < x) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}

Time Complexity: O(log n)

5. Summary

Algorithms are the heart of computer science. Good algorithm knowledge turns a normal coder into an efficient problem solver.
