Operators in Python

Operators let you perform computations and manipulate values. Python provides arithmetic, comparison, logical, bitwise, membership, identity, and assignment operators. Knowing operator behavior, precedence, and short-circuiting is crucial for correct logic.

Arithmetic operators

  • +, -, *, / (float division)
  • // (floor division), % (modulus), ** (power)

Comparison & logical

Comparisons: ==, !=, <, >, <=, >= return booleans. Logical operators and, or, not combine boolean expressions and use short-circuit evaluation (and stops at first False, or stops at first True).

Membership and identity

in checks membership in sequences/containers. Identity operators is and is not check object identity (same object in memory) — usually avoid for simple value equality.

Operator precedence

Precedence follows standard math: exponent > unary > *,/,% > +,- > comparisons > logical. Use parentheses for clarity.

Practical examples

a = 7
b = 3
print(a // b)   # 2
print(a % b)    # 1
print(2 ** 3)   # 8

# short-circuit
def expensive():
    print("called")
    return True

x = False and expensive()  # expensive() not called

Summary

Use operators idiomatically, avoid complex one-line expressions for readability, and prefer explicit parentheses when mixing logical/comparison operators.
