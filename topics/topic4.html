Operating Systems

1. Introduction

An Operating System (OS) is system software that manages computer hardware, software resources, and provides common services for applications. Examples include Windows, Linux, macOS, Android, and iOS.

2. Functions of an OS

  • Process Management
  • Memory Management
  • File System Management
  • Device Management
  • Security & Protection
  • User Interface

3. Types of Operating Systems

  • Batch OS
  • Multi-Tasking OS
  • Distributed OS
  • Real-Time OS
  • Mobile OS
