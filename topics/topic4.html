Control Flow — Conditionals

Control flow directs the order in which statements execute. The most common control constructs are conditionals (if/elif/else) and loops. Well-structured control flow yields readable, maintainable code.

If / elif / else

if condition:
    # branch A
elif other_condition:
    # branch B
else:
    # default

Use clear boolean expressions and guard clauses to handle error/edge cases early, reducing deep nesting. Avoid complex boolean expressions—break them into named variables for readability.

Example pattern — input validation

age_str = input("Age: ")
if not age_str.isdigit():
    print("Enter a valid number")
else:
    age = int(age_str)
    if age >= 18:
        print("Adult")
    else:
        print("Minor")

Exceptions vs conditionals

For operations that may fail (I/O, conversion), prefer exceptions for exceptional cases. Use try/except when the normal flow is the successful case and failures are rare.

Summary

Write simple, testable branches. Early returns (guard clauses) and small helper functions reduce complexity and make unit testing easier.
