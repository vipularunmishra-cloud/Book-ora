Loops — for & while

Loops repeat code blocks. Python offers for (iterate over sequences) and while (repeat while a condition holds). Use loops for traversal, aggregation, and repeated tasks. Prefer for for sequence iteration.

For loops and iterators

for x in [1,2,3]:
    print(x)

for i in range(5):    # 0..4
    print(i)

Python’s iterator protocol lets you iterate over any iterable (lists, tuples, generators, files). Generators (yield) provide lazy sequences that are memory efficient for large datasets.

While loop

n = 0
while n < 5:
    print(n)
    n += 1

Ensure termination conditions are reachable to avoid infinite loops. Use break to exit early and continue to skip to next iteration.

Comprehensions & functional alternatives

List/dict/set comprehensions and generator expressions replace many loop patterns with concise, readable constructs: 

squares = [x*x for x in range(10)]
even = [x for x in range(20) if x%2==0]

Best practices

  • Avoid heavy logic inside loops; extract to functions.
  • Prefer built-ins (sum, map, filter) for clarity/performance.
  • Profile hotspots when looping over large data.

Summary

Master loops and iterators — they’re fundamental. Learn generator patterns to process streams and large data efficiently.
